\doxysubsection{Detailed Description}
The \doxylink{classLeaf}{Leaf} class represents the end objects of a composition. A leaf can\textquotesingle{}t have any children.

Usually, it\textquotesingle{}s the \doxylink{classLeaf}{Leaf} objects that do the actual work, whereas \doxylink{classComposite}{Composite} objects only delegate to their sub-\/components.
